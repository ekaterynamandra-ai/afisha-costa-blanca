# Необходимые скиллы для проекта Афиша Costa Blanca

## Скиллы, которые нужно создать

### 1. `costa-blanca-scraper` — Парсер событий
**Зачем**: автоматический сбор событий из 7+ источников
**Что делает**:
- Парсит RSS-фиды (alicante.es, costablancaup.com, alicanteout.com)
- Работает с REST API (comunitatvalenciana.com, Eventbrite)
- Скрапит WordPress-сайты (hoyalicante.app, agendacultural)
- Дедупликация по title + date
- Сохраняет сырые данные в Supabase (status = 'raw')

**Технологии**: Python (requests, beautifulsoup4, feedparser) или Node.js (cheerio, rss-parser)

---

### 2. `costa-blanca-translator` — Переводчик + форматировщик
**Зачем**: превращение сырых данных на ES/EN в красивые русские посты
**Что делает**:
- Перевод ES/EN → RU через Claude API
- Генерация short_description (1-2 предложения для карточки)
- Генерация telegram_post по шаблону
- Оценка interest_score (1-10)
- Генерация практических советов (tips)
- Транслитерация для slug

**Промпт-шаблон для Claude API**:
```
Ты — редактор русскоязычной афиши Costa Blanca. 
Переведи и адаптируй это событие для русскоговорящих 
экспатов в Испании. Тон: дружелюбный, информативный, 
с практическими советами. Не "сухая" информация, 
а как друг рассказывает о крутом событии.
```

---

### 3. `costa-blanca-publisher` — Публикатор в Telegram
**Зачем**: автоматическая публикация одобренных событий
**Что делает**:
- Берёт approved события из Supabase
- Форматирует Telegram-пост (Markdown)
- Публикует через Telegram Bot API
- Обновляет telegram_published = true
- Воскресный дайджест "Что делать на выходных"

---

### 4. `costa-blanca-designer` — Дизайнер сайта
**Зачем**: создание и поддержка визуального стиля
**Что делает**:
- Генерация промптов для Lovable
- Создание UI-компонентов
- Цветовая палитра (средиземноморский стиль)
- Адаптивный дизайн
- Стиль карточек событий

**Палитра**:
| Токен | Цвет | Использование |
|-------|-------|--------------|
| Primary | `#1A365D` | Заголовки, навигация |
| Accent | `#F6AD55` | CTA, кнопки, акценты |
| Warm | `#E53E3E` | Горячие события, сердечки |
| Background | `#FFFAF0` | Фон страниц |
| Card | `#FFFFFF` | Карточки с тенью |
| Text | `#2D3748` | Основной текст |
| Muted | `#718096` | Вторичный текст |
| Success | `#38A169` | Бесплатно, доступно |

---

### 5. `costa-blanca-copywriter` — Копирайтер
**Зачем**: генерация и редактирование текстов
**Что делает**:
- Tone of voice: дружелюбный, как друг-экспат рассказывает
- Заголовки: короткие, цепляющие
- Описания событий: живые, с деталями
- Telegram-посты: эмоджи, структура, хэштеги
- Перевод специфической испанской лексики (fiestas, corrida, mercadillo и т.д.)

---

### 6. `web-scraper-reader` — Читалка веб-страниц
**Зачем**: считывание информации с сайтов-источников для анализа
**Что делает**:
- Открывает URL через Playwright/fetch
- Извлекает структурированные данные
- Сохраняет в JSON для дальнейшей обработки
- Работает с JavaScript-rendered страницами (SPA)

---

## Существующие скиллы, которые можно адаптировать

| Скилл | Как использовать |
|-------|-----------------|
| `tolik` (carousel) | Адаптировать для создания промо-картинок для Telegram/Instagram |
| `youtube-thumbnail` | Не нужен напрямую, но pipeline генерации изображений можно переиспользовать |

## Рекомендуемый порядок создания

1. **`costa-blanca-designer`** — сначала дизайн, чтобы видеть результат
2. **`costa-blanca-scraper`** — потом данные
3. **`costa-blanca-translator`** — потом обработка
4. **`costa-blanca-publisher`** — потом публикация
5. **`costa-blanca-copywriter`** — fine-tuning текстов
6. **`web-scraper-reader`** — по мере необходимости

## Внешние инструменты

| Инструмент | Зачем | Бесплатный план |
|-----------|-------|-----------------|
| Supabase | БД + Auth + Storage | Да, 500MB БД, 1GB Storage |
| Claude API | Перевод + обработка | Pay-per-use |
| Telegram Bot API | Публикация | Бесплатно |
| GitHub Actions | Cron-запуск скриптов | 2000 мин/мес бесплатно |
| Lovable.dev | Фронтенд | 1 проект бесплатно |
| Vercel (опционально) | Хостинг скриптов | Бесплатный план |
