# Контент-политика: Афиша Costa Blanca

## Золотое правило
**Ничего не публикуется без согласования с Катей.**

## Фильтры по интересам русскоязычной аудитории

### ПУБЛИКУЕМ (высокий интерес)
- Гастрономические фестивали (вино, еда, тапас, paella)
- Визуально красивые праздники (цветы, огни, костюмы, парады)
- Бесплатные семейные мероприятия
- Рынки и ярмарки (mercadillos, rastros)
- Природные события (цветение, миграция фламинго)
- Фейерверки и огненные шоу
- Пасхальные шествия (Semana Santa) — подборка по городам
- Спортивные события для зрителей (не участников)
- Тематические подборки ("Как празднуют Пасху на Costa Blanca", "Лучшие рынки апреля")

### НЕ ПУБЛИКУЕМ (низкий интерес для русскоговорящих)
- Фестивали испанской музыки (песни на испанском — не интересно аудитории)
- Кинофестивали (языковой барьер, нишевый контент)
- Длинные пешие маршруты (8+ км — слишком физически сложно)
- Узкоспециализированные культурные события (поэзия, театр на испанском)
- Религиозные мероприятия (кроме визуально зрелищных, как Semana Santa)

### ПУБЛИКУЕМ С ОСТОРОЖНОСТЬЮ
- Платные мероприятия — только самые яркие и уникальные
- Мероприятия далеко от побережья (>1.5 ч) — только если 9-10/10 по интересу
- Испаноязычные концерты — только если визуально/атмосферно (типа фламенко)

### Tier 3 (1-1.5 ч, только топ-события)
