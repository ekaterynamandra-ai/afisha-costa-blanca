# Правила контента v2 (обновлено по фидбеку)

## Структура публикаций

### Обязательная иерархия:
1. **Месячный дайджест по неделям** (в конце предыдущего месяца)
2. **Акцентные события** (детальные посты за 5-7 дней)
3. **Недельный дайджест** (суббота)
4. **Ежедневные рубрики** (с детьми, с собакой, подружки, гастро, природа, спорт)
5. **Подборка выходных** (среда)
6. **Напоминания** (день в день)

## Правила заголовков
- **Даты в заголовке!** Если событие привязано к конкретной дате — дата в заголовке
  - ДА: "🎮 18-19 АПРЕЛЯ: Expo Torrevieja + ComarCON"
  - НЕТ: "🎮 Expo Torrevieja + ComarCON" (дата внизу)
- Если регулярное/сезонное — без даты в заголовке ОК

## Ссылки
- **Каждое место** = Google Maps ссылка на КОНКРЕТНОЕ место (не координаты!)
  - ДА: `https://maps.app.goo.gl/XXXXX` (реальное место)
  - НЕТ: `https://maps.google.com/?q=38.123,-0.456` (просто точка)
- **"Смотри подробнее на сайте"** — для мест, которые требуют объяснения (Дюны, El Campico, Фламинго)
- **Ссылки на предыдущие посты** — если уже писали про место, добавлять "подробнее в нашем посте ↑"

## Гастро — ТОЛЬКО события
- НЕ просто "набережная Торревьехи"
- ДА: фестивали тапас, jornadas gastronómicas, дегустации вин, фестивали морепродуктов
- ДА: конкретные гастро-маршруты с датами (Tapa Cofrade, rutas de tapas)
- Если нет событий — не публиковать гастро-пост на этой неделе

## Природа / маршруты — КОНТЕКСТ
- Почему ИМЕННО в апреле? (погода +22, не жарко, цветение, фламинго)
- Что взять? (вода, бинокль, обувь)
- Сложность (лёгкий/средний)
- Что увидите (конкретно: цветы, птицы, виды)

## Спорт / активности — по сезону
- Весна: падель, теннис, велосипед, каяк
- Лето: море, SUP, снорклинг, прохладные напитки
- НЕ рекламировать частные бизнесы
- ДА: муниципальные корты, публичные маршруты, бесплатные активности
- При +40: НЕ предлагать падель/теннис/велик

## Google Maps — ПРАВИЛА
- Искать конкретное место по названию
- Формат: `https://maps.app.goo.gl/` или поиск по имени
- Перепроверять что точка ведёт на правильное место
- Palacio de los Deportes Torrevieja — это конкретное здание, не координаты

## Места требующие отдельного поста
Если упоминаем место впервые — нужен отдельный пост:
- Дюны Гуардамара (что это, почему интересно, как добраться)
- Рынок El Campico (что там, когда, лайфхаки)
- Салинас и фламинго (как смотреть, когда, откуда)
- Розовое озеро Торревьехи
После этого в подборках ссылаемся на этот пост.
