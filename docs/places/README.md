# База мест Costa Blanca

Здесь собираются ВСЕ интересные места для будущих подборок.
Цель: чтобы при создании поста не искать с нуля, а брать из готовой базы.

## Файлы

- **`walks.md`** — пешеходные маршруты, парки, природные зоны
- **`beaches.md`** — пляжи (с учётом сезона)
- **`markets.md`** — рынки (регулярные, блошиные, антикварные)
- **`food-festivals.md`** — гастро-события и фестивали
- **`festivals.md`** — праздники, фиесты, культурные события
- **`viewpoints.md`** — смотровые площадки, красивые виды
- **`with-dogs.md`** — места для собак
- **`with-kids.md`** — места для детей

## Формат записи

```markdown
### [Название места]

- **Где**: [город, район]
- **Координаты**: [Google Maps ссылка]
- **Расстояние**:
  - От Торревьехи: X мин
  - От Аликанте: X мин
  - От Бенидорма: X мин
  - От Гуардамара: X мин
- **Сезон**: [когда лучше посетить]
- **Что особенного**: [почему сюда стоит ехать]
- **Тип активности**: прогулка / фото / снорклинг / купание / велик
- **Подойдёт**: семьям / парам / с собакой / одному / компанией
- **Лайфхаки**: [что взять, лучшее время, парковка, итд]
- **Источник**: [где нашли, отзывы, ссылка]
- **Использовано в постах**: [список ID постов]
```

## Правила

1. Места проверяем перед добавлением (Google Maps, отзывы, актуальность)
2. Каждое место = отдельный раздел в файле
3. Если место в нескольких постах — обновлять "Использовано в постах"
4. Не дублировать — перед добавлением искать по названию
