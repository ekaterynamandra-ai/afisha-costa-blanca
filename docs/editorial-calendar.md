# Редакционный календарь: логика публикаций

## Правило тайминга
- **Месячный обзор**: последняя пятница предыдущего месяца
- **Недельный дайджест**: суббота (на следующую неделю)
- **Подборка выходных**: среда (на ближайшие Сб-Вс)
- **Акцентные события**: за 3-7 дней до события
- **Регулярные рубрики**: фиксированные дни (рынки — Ср, с собакой — Чт, и т.д.)
- **Сезонные**: по ситуации, но минимум 1 в неделю

## Минимум 1 пост в день (7-14 в неделю)

## Типовая неделя (шаблон)

| День | Тип поста | Пример |
|------|----------|--------|
| **Сб** | Дайджест на следующую неделю | "📋 Неделя 14-20 апреля: что интересного" |
| **Вс** | Сезонное / природа / тематическое | "🦩 Фламинго: пик сезона!" |
| **Пн** | Акцентное событие недели | "🎭 Moros y Cristianos через 4 дня!" |
| **Вт** | Подборка по аудитории (ротация) | "👨‍👩‍👧 С детьми: 5 идей на неделю" |
| **Ср** | Подборка выходных + рынки | "🌟 Выходные: куда пойти" |
| **Чт** | Регулярная рубрика (собаки/гастро/природа) | "🐕 С собакой: новый маршрут" |
| **Пт** | Напоминание о выходных + рынок | "🛍️ Завтра мега-рынок Торревьехи!" |
