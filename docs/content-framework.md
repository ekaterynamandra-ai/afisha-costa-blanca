# Контент-фреймворк: типы публикаций

## Иерархия контента

```
МЕСЯЦ (апрель)
  ├── Календарь месяца (все события + праздники + выходные)
  ├── Большие праздники (Semana Santa, San Juan, Navidad...)
  │     ├── Обзор: где и что празднуют по городам
  │     └── Акцент: самое яркое празднование + детали
  ├── Топ-события месяца (3-5 самых ярких)
  │
  НЕДЕЛЯ
  ├── Дайджест недели (понедельник утро)
  ├── Подборка выходных (пятница вечер)
  │
  ПОДБОРКИ ПО АУДИТОРИИ
  ├── 👨‍👩‍👧 С детьми на этой неделе
  ├── 🐕 С собакой: куда пойти
  ├── 👯‍♀️ С подружками
  ├── 💑 Вдвоём / романтика
  ├── 🏃 Активный отдых / природа
  ├── 🍽️ Гастро-подборка
  ├── 📸 Самое фотогеничное
  ├── 🆓 Бесплатное
  │
  ТЕМАТИЧЕСКИЕ
  ├── 🌸 Сезонное (цветение, фламинго, пляжный сезон)
  ├── 🛍️ Рынки: когда, где, что (расписание)
  ├── 🏖️ Пляжи: новости, открытия, условия
  └── 🌡️ Погода + что делать в такую погоду
```

## Типы постов

### 1. КАЛЕНДАРЬ МЕСЯЦА
**Когда:** 1-е число месяца (или за 1-2 дня)
**Формат:** Список всех ключевых дат + праздников + выходных
**Пример заголовка:** "📅 АПРЕЛЬ 2026: все важные даты Costa Blanca"

### 2. ОБЗОР БОЛЬШОГО ПРАЗДНИКА
**Когда:** За 5-7 дней до праздника
**Формат:** Где празднуют + расписание по городам
**Пример:** "🕊️ SEMANA SANTA: где и когда смотреть процессии"

### 3. АКЦЕНТ / ЛУЧШЕЕ
**Когда:** За 3-5 дней до события
**Формат:** Одно конкретное событие — подробно
**Пример:** "🏆 Самая зрелищная Semana Santa — Орихуэла"

### 4. ДАЙДЖЕСТ НЕДЕЛИ
**Когда:** Понедельник утро
**Формат:** 5-7 событий на неделю
**Пример:** "📋 НЕДЕЛЯ 6-12 АПРЕЛЯ: что интересного"

### 5. ПОДБОРКА ВЫХОДНЫХ
**Когда:** Пятница вечер
**Формат:** 3-5 событий на Сб-Вс
**Пример:** "🌟 ВЫХОДНЫЕ 12-13 АПРЕЛЯ: куда пойти"

### 6. ПОДБОРКА ПО АУДИТОРИИ
**Когда:** 1-2 раза в неделю, ротация аудиторий
**Формат:** 3-5 событий для конкретной аудитории
**Пример:** "🐕 С СОБАКОЙ: 5 мест на эту неделю"

### 7. ТЕМАТИЧЕСКАЯ ПОДБОРКА
**Когда:** По ситуации
**Формат:** Глубокий обзор одной темы
**Пример:** "🦩 Фламинго: где смотреть, когда ехать, что брать"

### 8. ОДИНОЧНОЕ СОБЫТИЕ
**Когда:** За 1-3 дня до события
**Формат:** Детальный пост об одном ярком событии
**Пример:** "🎭 Moros y Cristianos — юбилей 750 лет в Алькое"

### 9. РАСПИСАНИЕ РЫНКОВ
**Когда:** Раз в месяц (обновление) + напоминание в день рынка
**Формат:** Таблица: день — город — тип — часы
**Пример:** "🛍️ РЫНКИ АПРЕЛЯ: расписание по дням"
