## Правила (короткая выжимка из README)

- **Каждое интересное событие — отдельный event_alert пост за 1-3 дня** (для TG)
- Фильтр: ✅ pool parties с атмосферой / world music / закрытые ежегодные с историей / fiestas с реконструкцией. ❌ просто диджей в клубе / банальные beach club сеты
- Не дублировать — если событие было в посте, повтор только с новым углом
- См. memory `feedback_kostoplan_festivals.md` для расширенной версии правил
