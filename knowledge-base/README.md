# Knowledge Base — Costa Blanca / Kate in Spain

Общая база знаний по всему интересному что мы с Катей находим и видим лично:
- **События** (фестивали, fiestas, концерты, разовые мероприятия)
- **Места** (пляжи, городки, природа, viewpoints, тропы)
- **Гастро** (рынки, кафе, кулинарные традиции — массовое, не реклама частников)
- **Опыт** (туры, экскурсии, активности, традиции)

Растёт со временем. **Каждая новая находка — сразу сюда**, не ждём пока пригодится.

## Структура

```
knowledge-base/
├── README.md          ← этот файл
├── events.md          ← всё про события, по дате + по типу

```

## Откуда тянется и куда уходит

**Источники наполнения:**
- Личный опыт Кати (поехала-увидела-понравилось)
- Личный опыт мужа/друзей
- Ресёрч через web (когда я ищу инфо для постов или статей)
- Случайные находки в Instagram / Telegram / форумах
- Подсказки подписчиков

**Куда уходит контент из базы:**
- **Telegram-канал @kate_in_spain** — еженедельные посты, event-alerts (см. `content/schedule/`)
- **Сайт kateinspain.es** — разделы Жизнь (места) и Переезд/Бизнес (полезные гайды) (см. `site/src/content/articles/`)
- **Сторис / Reels** (на будущее, когда подключим)

## Правила добавления

1. **Лучше избыточно, чем пропустить.** Сомневаешься «нужно ли это» — добавляй с пометкой `??`. Потом разберёмся.
2. **Каждая запись — минимум:** название, где, когда (если применимо), краткое «зачем сюда идти», ссылка на источник.
3. **Личное мнение Кати** — выделять отдельным блоком `> Катя:`. Это самая ценная часть базы.
4. **Не дубль контент из самих постов** — здесь reference, а не пересказ. Ссылка на пост в TG / на статью на сайте.
5. **Не реклама частного бизнеса** (правило проекта: запрет на отели/рестораны/частные школы). ОК: массовые attractions (Terra Mítica, Aqualandia, SHA, Achill, Benidorm Palace), культурные fiestas, природные места, рынки.

## Как пользоваться при планировании

**При планировании TG-недели:**
1. Открыть `events.md` → проверить «что в ближайшие 7-10 дней»
2. Включить интересное как event_alert посты
3. Подмешать в weekend-подборки если умеренно

**При написании статей сайта:**
1. Раздел Жизнь → тянем из `places.md` (детальные гайды по местам)
2. События для сайта → тянем из `events.md` (evergreen описания)
