# /plan

Создаёт план Костоплана на заданный период. Разбивает его на недели (пн-вс) и генерирует скелеты `week*.json` в `content/schedule/` — с датами, временем, типом и заголовком, но без полного текста.

## Использование

- `/plan 2026-04-22 2026-05-31` — план с Ср 22 апреля по Вс 31 мая
- `/plan week 5` — одна конкретная неделя
- `/plan next month` — следующий месяц

## Что делает

1. Читает:
   - `docs/editorial-calendar.md` — логика тайминга
   - `docs/content-framework.md` — типы постов
   - `docs/content-policy.md` — что публикуем
   - `content/events/events-*.json` + `content/data/*.md` — события с датами
2. Разбивает период на недели пн-вс
3. Для каждой недели создаёт `content/schedule/weekN-<от>-<до>.json` с `approved: false` и постами в виде скелетов:
   ```json
   {"id": "w5-mon-1", "date": "2026-04-27", "time": "09:00", "title": "📋 Дайджест недели", "type": "weekly_digest", "status": "draft", "photo": "", "text": ""}
   ```
4. Применяет ритм: 1 пост/день по умолчанию, +1 на знаковые дни (праздники, большие ивенты, выходные перед ними)
5. Обновляет `content/schedule/plan-<период>.md` — таблица-оверхед, что где
6. Показывает сводку: сколько недель, сколько постов, главные акценты

НЕ пишет тексты — для этого `/week`.

## Ритм по умолчанию (1 пост/день)

| День | Тип | Время |
|------|------|-------|
| Пн | weekly_digest | 09:00 |
| Вт | audience_* или place_intro | 10:00 или 18:00 |
| Ср | weekend_picks | 10:00 |
| Чт | рубрика (рынки/собака/гастро) | 18:00 |
| Пт | reminder_market / today_event | 08:00 или 10:00 |
| Сб | today_event / places_collection | 14:00 |
| Вс | nature / sunset / month_recap | 10:00 |

## Когда второй пост в день

- Праздничный выходной (Pascua, 1 мая, San Juan) — `today_holiday` утром + `audience_free` вечером
- Крупное событие в день (Moros y Cristianos Alcoy, Santa Faz) — `today_event` + атмосферная рубрика
- Последняя пятница месяца — дайджест следующего месяца в 19:00 добавляется к утреннему посту

## Контент-фильтр (всегда применять)

- ❌ Испанские музыкальные/кино фестивали — не интересны
- ❌ Походы 8+ км без остановок
- ❌ Платные события кроме топовых
- ✅ Гастро, парады, бесплатные семейные, природа, рынки
- Приоритет локаций: Гуардамар → Торревьеха → Эльче → Аликанте → дальше

## После /plan

1. Пересмотреть скелеты в week*.json — убрать лишнее, добавить недостающее
2. `/week N` — развернуть одну неделю в полные тексты
3. `/preview` — HTML-превью для просмотра
4. `/approve N` — одобрить для автопубликации
